Introduction to HTML

HTML
HTML, an abbreviation for HyperText Markup Language, is the programming language that allows developers to create the elements on websites. For example, you are reading html right now! I am using a Code Editor to edit the contents of this HTML file, and seeing it in the browser. Here is what I am looking at whilst writing this:
Syntax
HTML has a pointy look to it due to it's syntax. It is reliant on spacing to make the code look organized and readable.

Web Development
HTML is also often seen with other website (also referred to as front-end) programming languages like Cascading Style-Sheets (CSS) and JavaScript (JS). CSS makes your HTML look beautiful, and JS makes your HTML do stuff (functionality).

Syntax Continued
If you look at the screenshot that I included earlier, you will notice that my HTML document (and all other HTML documents) will have something in common. Each red word that you see is called a tag. A tag lets the browser running the HTML know that that specific tag, and the contents inside of it, are different, and have different functions. For example, the <button> tag tells the browser to get the automatic style for it. This is so that the button appears on screen, and looks specific to each browser. Eventually, when you learn JavaScript, you will learn how to add functionality to your web page.

First HTML

Writing HTML
Now that you know what HTML is and where it can be used, it is time to start coding in HTML. First, you need to install a code editor of your choice. Here are a few good ones:
  1. Visual Studio Code
  2. Atom
  3. Sublime Text
  4. Notepad++
If you decide to use Visual Studio Code, you should download the extension called "Live Server", created my Microsoft. This allows the browser to automatically refresh with the new changes once you have saved your HTML. If you are not using Visual Studio Code, do not worry. You will just need to save your code, then refresh once you get to the browser.

Your First HTML
In your code editor, go to file > new, and creating a file ending in .html. In most cases, you should name your file: "index.html". This is not necessary, but is a naming standard. This is how web hosting services, like GitHub know that this is the default/root file. Now, in that document, write the following contents: 
<!DOCTYPE html>
<html>
    <body>
        <h1>Welcome to my first web page!</h1>
    </body>
</html>

Once you have inserted this code, press "ctrl" then "s" to save the file. If you are in visual studio code, right click on the file (if you have Live Server installed), then click "Open with Live Server".
